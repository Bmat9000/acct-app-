# R6 Account Manager

## v1.0 — Initial Release

- Master password setup and encryption on first launch
- Save and manage multiple R6 smurf accounts
- Store email, email password, Ubisoft username, Ubisoft login email, and Ubisoft password per account
- Custom account names with auto-numbering (Account 1, Account 2, etc.)
- Description field for notes on each account
- Show/hide toggle on all password fields
- Copy to clipboard button on all fields
- Dark themed desktop UI
- All data stored locally and encrypted on your PC
